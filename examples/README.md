# 📂 案例 / Examples

真实的端到端使用案例，展示核心部各小队处理完整委托的全流程。

| # | 案例 | 委托内容 | 涉及部门 | 复杂度 |
|---|------|---------|----------|--------|
| 1 | [竞品分析](competitive-analysis.md) | "分析 CrewAI vs AutoGen vs LangGraph" | 中书→门下→财务小队+战斗小队+书记小队 | ⭐⭐⭐ |
| 2 | [代码审查](code-review.md) | "审查 FastAPI 代码的安全性" | 中书→门下→战斗小队+审判小队 | ⭐⭐ |

每个案例包含：
- 📜 **任务**：原始指令
- 📋 **策划部规划**：任务拆解方案
- 🔍 **监察部审议**：审核意见（含驳回/返工记录）
- ⚔️ **各部执行结果**：每个部门的产出摘要
- 📮 **最终战报**：调度部汇总的完整报告

> 💡 这些案例基于真实运行记录整理，任务 ID 和时间戳已脱敏。
